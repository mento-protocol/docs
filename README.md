# Mento

Pools that discover price on-chain have a cost: LPs and traders lose to arbitrageurs. For foreign exchange, the rate already exists—there’s no need to pay that cost. Mento uses the **existing** price so you can swap at the rate. On-chain FX can compete with off-chain.

**What Mento offers:** Fixed-price market makers (FPMMs) for stablecoins and FX on Celo and other chains. Swap at the oracle rate. When the pool’s composition drifts, allowlisted strategies rebalance it. No curve slippage, no structural transfer to arbitrageurs.
