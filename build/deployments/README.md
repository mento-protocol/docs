# Deployments

**Who this is for:** Integrators and deployers who need contract addresses, deployment parameters, and verification steps for Mento V3 (and legacy deployments) on supported chains.

Addresses are organized by **chain ID** and **deployment name**:

| Chain | Chain ID | Deployment(s) |
|-------|----------|----------------|
| **Celo Mainnet** | 42220 | `mainnet` — FPMM, Router, Reserve, CDP (GBPm), oracles, stable tokens |
| **Celo Sepolia** | 11142220 | `testnet-v2-rc5` — V3 + legacy (Broker, BiPoolManager), test tokens and relayers |
| **Monad** | 143 | `mainnet`, `monad-mainnet` — stable tokens (USDm, GBPm, StableTokenSpoke) |
